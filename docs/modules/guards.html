getsType (ty, t)
Type-checking errguards. guardia.guards automatically creates functions such as getsNumber or getsNil for all Lua base types. It also supports the __type metafield for custom types.

Parameters:

  • ty string Type to check for.
  • t function or nil Type function to use.
