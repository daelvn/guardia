Module guardia.init

Guards for Lua.

Based upon guard.lua. This version is rather thought for curried functions and use with MoonScript.

Guard (filter, transform, f, ...)
guard.lua-style Guard. Curried function.

Parameters:

  • filter function Filters whether the guard is applicable or not.
  • transform function If the guard is applicable, apply it and return.
  • f function Function to be guarded.
  • ... Arguments to pass to f

Returns:

    Result of f or nil, if guarded.

Biguard (filter, transform, antitransform, f, ...)
Guard with two transformers, one if the filte passes, one if it doesn't. Curried function.

Parameters:

  • filter function Filters whether the guard is applicable or not.
  • transform function If the guard is applicable, apply it and return.
  • antitransform function If the guard is not applicable, apply this one and return. Passed with last arg.
  • f function Function to be guarded.
  • ... Arguments to pass to f

Returns:

    Result of f.
